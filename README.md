💻 System Optimizer v3.2

System Optimizer v3.2 is a lightweight console-based system optimization tool compiled as a standalone .exe executable for Windows.

Built using C++ for maximum performance and low-level system access, this application focuses on delivering efficient resource management and system maintenance operations with minimal overhead.

🚀 Features
1️⃣ Temporary Files Cleaner

Removes unnecessary temporary files generated by the operating system and applications to free up disk space.

2️⃣ PC Performance Booster

Optimizes background processes to improve overall system responsiveness and speed.

3️⃣ Windows Registry Repair

Scans and fixes invalid or corrupted registry entries to enhance system stability.

4️⃣ RAM Optimization

Releases unused memory to improve performance during demanding tasks.

5️⃣ Disk Defragmentation

Reorganizes fragmented files on HDD drives to improve data access speed.

6️⃣ Exit

Safely closes the application.
